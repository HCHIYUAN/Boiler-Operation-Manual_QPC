所要 30~40 分
!
複数台制御使用者必読｜MULTI-UNIT CONTROL
使用多機控制盤時，実施前に「台数制御」設定を「×」（経路：MENU → 設定 → 台數制御）。全ブロー完了後は必ず「○」，そうしないと複数台制御が正常動作しません。
A
圧力判断と排出開始
PRESSURE CHECK & DRAIN START
關鍵圧力區
0.05 ~ 0.2 MPa
A1PREP
ブロー管末端固定確認
確認ブロー管末端確實固定於水槽。未固定會因反作用力飛跳造成燙傷。
管路末端有金屬夾固定，伸入水槽內
A2PRESSURE CHECK
圧力確認 0.05 ~ 0.2 MPa
最有效的ブロー圧力區間。圧力過低時需暫時啟動ボイラー升壓至 0.1 MPa。
圧力錶指針安定黄色範囲内
A3IF LOW PRESSURE
圧力 < 0.05 MPa 対処手順
圧力過低時依上圖三步驟処理：運転 ON → 升壓至 0.1 MPa → 運転 OFF → 回ステップへ A4。
圧力安定後再開始ブロー，避免無壓状態下的ブロー效果不佳
A4CRITICAL
ボイラーブロー弁ゆっくり開
確認ブロー管末端安全後，緩慢將ボール弁ハンドルを水平。突然打開會造成熱水大量噴出。
ハンドル平行於管路，ブロー管有水流入水槽
B
排出待ちと給気加速
WAIT & VENT ACCELERATION
等待時間
10 分以上
B1WAIT
静置待機 10 分
讓有壓鍋水自然排出。圧力計を観察，指針低下待ち 0 MPa。
圧力計指針低下 0 MPa（青範囲）
B2CRITICAL
圧力 0 MPa 後緩慢開給気弁
圧力低下 0 MPa 後，緩慢打開給気弁加速ブロー。突然打開會讓蒸気快速噴出。
ハンドル轉到水平，ボイラー內水加速排出
C
リセットと運転復帰
RESET & RESTART
完了階段
計 3 步驟
C1RESET
【全ブロースイッチ】押下リセット
ブロー完全終了後，パネルの全ブロースイッチ。ボイラー表示ランプ・全ブローランプ・通知ランプ全消灯。
全表示ランプ消灯，ブザー停止
C2CLOSE VALVES
關閉給気弁 + ボイラーブロー弁
ボール弁2個全閉（ハンドル管路に垂直）。再確認後回ステップへ。
兩弁ハンドル垂直，水/ガス漏れなし
C3MULTI-UNIT ONLY
複数台制御設定を「○」
僅多機控制使用者執行。將「台數制御」從「×」復帰「○」。復帰しないと複数台制御動作不可。設定経路：MENU → 設定 → 台數制御 → 選 ○ → SET。
パネル表示「台数制御 ○」
